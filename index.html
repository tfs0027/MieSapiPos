🖨️ Koneksi Printer Thermal
📡 Web Bluetooth API
Browser akan mencari printer Bluetooth ESC/POS di sekitar. Pastikan printer dalam kondisi pairing mode dan browser kamu mendukung Web Bluetooth (Chrome/Edge desktop).
📶 WiFi / Network Printer
Printer thermal yang terhubung ke jaringan WiFi yang sama. Biasanya menggunakan port 9100 (RAW) atau 80 (HTTP).
🔌 USB Serial Port
Menggunakan Web Serial API (Chrome/Edge). Colokkan printer via USB lalu klik tombol di bawah.
🌐 ESC/POS HTTP Bridge
Mengirim perintah ESC/POS ke server lokal (mis. node-escpos, QZ Tray, atau PrintNode) yang terhubung ke printer fisik.
